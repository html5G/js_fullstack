git 命令
git init 为版本管理
git add readme.txt 添加一个readme.txt文件
git commit -m “wrote a readme file”确认添加的文件
git status查看修改的文件、查看文件的状态
git diff readme.txt查看上一次修改的地方
git log 查看日志
git log --pretty=oneline 对日志进行筛选
git reset --hard HEAD^ 回到上一个版本，几个"^"就回到几个版本
git reset --hard ID所要回到的版本（ID是所要版本的ID 是前五位）
